# 📘 ACCA Landing Page UI (React Project)

## 🚀 Project Overview

This project is a responsive **ACCA Landing Page UI** built using **React.js** and **Bootstrap**. It replicates a modern educational landing page with multiple sections and a functional modal.

The focus of this project is on **UI design, component structure, and React concepts**.

---

## 🛠️ Tech Stack

* ⚛️ React.js (Vite)
* 🎨 Bootstrap
* 🧠 React Hooks (`useState`)
* 💻 JavaScript (ES6+)

---

* Contains form inputs:

  * Phone Number
  * Email
  * Qualification
  * Interest
* Close button available
* Shows success alert on submit

## ⚙️ Installation & Setup

Follow these steps to run the project locally:

```bash
# Clone the repository
git clone https://github.com/aniljiA1/anilnxtwaveproject8.git

# Navigate to project folder
cd acca-ui

# Install dependencies
npm install

# Start development server
npm run dev

http://localhost:5173/
```
